Interactive Notation

Creating a genuine interface for notational interaction is a key feature enabling for authentic engagement with music learning through multimedia. The rationale and design of this feature is discussed in detail in the Design Choices page.

Creation Workflow

Through discussions with colleagues, several ideal features were identified which would allow for efficient creation of music education multimedia content.

  • Importation of MusicXML (notation format) rather than images for notation
  • Editable settings for the display of notation (e.g. sections, length, animation, MIDI playback)
  • Ability to use a keyframe system to add notation over footage, as well as control parameters such as highlighting and focus areas.
  • Audio mixing control for MIDI and in-video sound.
  • A simple interactive editor which can utilise sections of imported notation or be used to build notation directly within the editor. o This should also allow for creation of instant feedback for answers such as common wrong answers or the correct answer.
  • Ability to export notation inputted by students (e.g. student writes a bar of music inbetween two existing bars) for further direct feedback from teachers.

As mentioned in our goals, increasing accessibility to the creation of multimedia via the platform was a must (See page: Goals). This need for the proposed platform to allow a high level of control over the multimedia content while keeping the creative process and workflow as simple as possible was supported through the discussions I had with my colleagues.
